ColorfulZz.ColorTable — Method
ColorTable(lut)

Builds a color table (a.k.a. look-up table) from a vector lut of colorant (as defined in ColorTypes.jl). lut could be created with the package ColorSchemes.jl (see example below).

using ColorSchemes
ColorTable(colorschemes[:jet])
